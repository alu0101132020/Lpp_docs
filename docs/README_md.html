    Práctica de laboratorio de LPP - Grado en Ingeniería Informática, ULL.¶ ↑

    Para instalar esta gema, deberemos efectuar en consola el siguiente comando.

    $ gem install tdd

    Usage¶ ↑

    Las nuevas funciones nos permiten calcular la huella nutricional de los alimentos, utilizar las funciones de los arrays para obtener información de los platos, como el plato con mayor huella nutricional… Y además con la función precio_max podremos modificar el vector de precios para obtener un vector con precios cambiados en base al plato con mayor huella nutricional.
